React JS

What is React?

React is a JavaScript library designed for building user interfaces of web applications. Its key concepts include components, virtual DOM, and JSX.

Key Principles of React

React is based on several key principles, such as declarativity, component-based architecture, and universality.

Declarativity in React

Declarativity in React means that you describe what should be displayed on the screen, rather than how it should be achieved.

const App = () => {
          return (
            <div>
              <h1>Hello, world!</h1>
            </div>
          );
        };

Component-Based Architecture

In React, everything is built on components, which are independent, reusable blocks of interface.

class Welcome extends React.Component {
  render() {
    return <h1>Hello, {this.props.name}</h1>;
  }
}

Algorithm and Virtual DOM

React utilizes a virtual DOM and an efficient updating algorithm for maximum performance when updating the interface.

JSX in React

JSX allows embedding any valid JavaScript expressions inside curly braces.

const element = <h1>Hello, World!</h1>;

Attributes in JSX

To use a string literal as an attribute value, quotes are used.

If a JavaScript expression is needed in the attribute value, curly braces are used:

// String literal as attribute value
const element1 = <img src="react-logo.png" alt="React Logo" />;

// JavaScript expression in attribute value
const imageUrl = 'react-logo.png';
const element2 = <img src={imageUrl} alt="React Logo" />;

Security and Babel Compilation

User-entered data can be safely used in JSX, as it prevents code injection attacks.

Babel compiles JSX into React.createElement() calls, creating React elements.

Advantages of Using React

React offers several advantages over other approaches to interface development, such as ease of use, high performance, and a wide range of ecosystem capabilities.
